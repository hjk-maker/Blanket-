What is Blanket?

Blanket is a modular Python architecture combining safe website image ingestion, pattern extraction, and meta‑reasoning. It is built for long‑term research and transparency — not hidden automation.

Core Capabilities

  • Website image cloning with validation & limits
  • Pattern normalization for future ML pipelines
  • WHY–WHY introspection (reasoning about reasoning)
  • Persistent consciousness memory (JSON)
  • Thread‑safe graphical interface

Ethical by Design

Blanket avoids unsafe behaviors by default:

  • No hidden training or scraping
  • No system‑level execution
  • No silent network actions
  • robots.txt support for public hosting

Future Vision (2027‑Ready)

  • Vision Transformers
  • Self‑adapting UI layers
  • WHY‑graph visualization
  • Reasoning engines & symbolic memory
